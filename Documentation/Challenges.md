# Challenges

## Challenge 1
Integrating customer engagement data coming from multiple business domains.

Solution:
Unified ingestion using Dataflow Gen2 and OneLake.

------------------------------------------------

## Challenge 2
Maintaining high-quality curated datasets.

Solution:
Implemented Bronze → Silver → Gold Medallion Architecture.

------------------------------------------------

## Challenge 3
Supporting both batch analytics and streaming analytics.

Solution:
Implemented Lakehouse for batch processing and Eventstream + Eventhouse for real-time ingestion.

------------------------------------------------

## Challenge 4
Providing business-ready reporting.

Solution:
Created Warehouse views and Semantic Model for Power BI.

------------------------------------------------

## Challenge 5
Supporting scalable analytics.

Solution:
Used Delta Lake tables with Fabric's scalable Spark engine.
